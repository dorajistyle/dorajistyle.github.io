[무료램디스크] 무료 램디스크로 컴퓨터 속도를 빠르게!(Dataram RAMDisk)


Dataram RAMDisk - 무료 램디스크로 컴퓨터 속도를 빠르게!

어떻게 하면 컴퓨터를 조금이라도 빠르게 쓸 수 있을까요?
그 방법 중 하나가 램디스크를 이용하는 것입니다.
메모리는 하드디스크보다 읽고쓰는 속도가 현저하게 빠릅니다.
평소에 사용되지 않는 메모리 용량을 나누어 램디스크로 활용하는 것이죠.


그럼 램디스크를 활용해서 컴퓨터 속도를 올려볼까요?
무료 램디스크인Dataram RAMDisk는 4G의 디스크 용량까지 무료로 사용 할 수 있어요.
저에겐 충분하고도 넘치는 용량이죠.
우선 램디스크를 설치 하고, 원하는 만큼 용량을 설정해 주세요.
저는 2G의 메모리를 사용하고 있기 때문에 128M만 할당을 해 주었습니다.
이 정도로도 충분히 효과를 볼 수 있어요.
램디스크 드라이브가 F로 잡혔군요.
이건 사용하는 컴퓨터의 파티션이 어떻게 나뉘어 있느냐에 따라 할당 문자가 달라요.

램디스크를 활용하는 보편적인 방법은 세 가지가 있습니다.

  • 시스템 임시 폴더를 램디스크로 변경하여 컴퓨터 속도를 올리는 방법
  • 인터넷 임시 폴더를 램디스크로 변경하여 컴퓨터 속도를 올리는 방법
  • 자주 쓰는 프로그램의 임시 폴더를 램디스크로 변경하여 속도를 올리는 방법

저는 시스템 임시폴더와, 인터넷 임시 폴더만 램 디스크로 설정 해 주었어요.
128메가로는 다른 프로그램의 임시 폴더까지 처리하기 벅차니까요.^^;





임시 폴더 옮기기 - 무료 램디스크로 컴퓨터 속도를 빠르게!

시스템 임시 폴더를 램디스크로 변경하여 컴퓨터 속도를 올리는 방법
  1. WIN(윈도우키)+브레이크(Pause Break) 키를 눌러 시스템 정보를 봅니다.
  2. 고급 시스템 설정으로 들어갑니다.
  3. 환경 변수(Environment Variables)를 클릭합니다.
  4. 사용자 변수(User variables)와 시스템 변수(System Variables)에 있는 TEMP와 TMP의 경로를 램 디스크의 켱로로 변경 해 줍니다.

인터넷 임시 폴더를 램디스크로 변경하여 컴퓨터 속도를 올리는 방법




익스플로러 인터넷 임시 폴더 옮기기 - 무료 램디스크로 컴퓨터 속도를 빠르게!

익스플로러 인터넷 임시 폴더를 램디스크로 변경하여 컴퓨터 속도를 빠르게!
  1. 익스플로러에서 인터넷 옵션으로 들어갑니다.
  2. 일반(General) - 브라우징 히스토리(Browsing history) 의 설정(Settings)을 클릭합니다.
  3. 캐시 용량(diskspace to use)을 램 디스크 용량과 같거나 적게 설정해 주고, 폴더 옮기기(Move folder)를 눌러 램디스크로 임시 폴더를 옮겨줍니다.


크롬 인터넷 임시 폴더 옮기기 - 무료 램디스크로 컴퓨터 속도를 빠르게!


크롬 인터넷 임시 폴더를 램디스크로 변경하여 컴퓨터 속도를 빠르게!
  1. 크롬 바로가기를 우클릭 해서 속성을 누릅니다.
  2. 대상(Target)의 맨 뒤에 --disk-cache--dir=램디스크 드라이브(F:)를 넣어줍니다.



파이어폭스 인터넷 임시 폴더 옮기기 - 무료 램디스크로 컴퓨터 속도를 빠르게!


파이어폭스 인터넷 임시 폴더를 램디스크로 변경하여 컴퓨터 속도를 빠르게!

  1. 파이어 폭스 주소창에 about:config를 치고 엔터를 누릅니다.
  2. 환경 설정창이 뜨면 browser.cache.disk.parent_directory를 찾습니다.
  3. 설정이 없다면 설정을 새로 만들어주고 값으로 램 디스크 드라이브를 넣어 줍니다.
파이어 폭스의 경우는 램 디스크를 사용할 수도 있지만,
직접 메모리를 캐쉬로 사용하는 방법이 있어요.
잘 설명된 사이트가 있어서 링크 합니다.
